## Hi there 👋
# Anel Zhunussova

Machine Learning Engineer | Neuroimaging Data Scientist | AI for Biomedical Research

I develop computational pipelines for analyzing high-dimensional biomedical data, with experience spanning clinical machine learning, neuroimaging analysis, and AI-driven biomedical knowledge synthesis. My work focuses on building reproducible analytical workflows that bridge research data and predictive modeling.

My background includes research collaborations with **Harvard Medical School / Massachusetts General Hospital** and the **Center for Neuroscience Imaging Research (IBS / SKKU)**, where I developed machine learning models for clinical and neuroimaging datasets.

---

## Research & Engineering Projects

### Clinical-Machine-Learning-Pipeline
Machine learning framework for analyzing multidimensional clinical and behavioral datasets.

Key features:
- Modular R-based pipeline for clinical ML workflows
- Nested cross-validation for unbiased model evaluation
- Explainable AI using SHAP for feature interpretation
- Applied to vaccine hesitancy prediction using clinical, psychological, and sociodemographic data

Tech stack: R, machine learning, statistical modeling

---

### Neuroimaging-Analysis-Pipeline
MATLAB pipeline for brain–behavior prediction from time-resolved fMRI data.

Key features:
- Modular neuroimaging workflow for behavioral alignment, denoising, and feature extraction
- Dual-stream modeling architecture integrating:
  - HRF-based activation features
  - Dynamic Conditional Correlation (DCC) connectivity features
- Predictive modeling using cross-validated principal component regression

Tech stack: MATLAB, CANlab, fMRI analysis, time-series modeling

---

### Biopharma-Document-RAG-Pipeline
Experimental Retrieval-Augmented Generation (RAG) framework for analyzing biomedical and regulatory documents.

Key features:
- Hybrid retrieval combining dense and sparse search
- Evaluation of hybrid vs vanilla RAG architectures
- Applied to biomedical literature and regulatory drug documentation
- Designed for knowledge synthesis in drug development and regulatory analysis

Tech stack: Python, LLMs, FAISS/BM25 retrieval, RAG systems

---
